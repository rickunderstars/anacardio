AnaCardio

AnaCardio is a prototype visualization tool for the Biosense Webster Triangulated Mesh file format 2.0.

This project is part of Riccardo Bovo's bachelor degree thesis in Computer Science of at the University of Milan (Università degli Studi di Milano), under the supervision of Prof. Marco Tarini and Dr. Massimo Walter Rivolta.

The source code is available here.
